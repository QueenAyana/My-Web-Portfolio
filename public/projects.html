NYT Artical Scraper

The NYT React Article Scrubber is a web application that captures the headline from a users search query. Users have the ability to save articles for later reference, as well as delete old articles they no longer wish to keep. NYT React Article Scrubber demonstrates CRUD, four basic functions of persistent storage, using MongoDB. You can view the github repo here!

Applitcation made with:

  • React.js
  • ES6 coding format
  • Bootstrap CDN
  • CSS
  • MongoDB/Mongoose-NPM
  • Node.js
  • Express-NPM
  • Axios-NPM
